## Initial Request

The session began with a request to build an expandable browser game prototype called **Reactor Logic: Foam Run** using React, TypeScript, Three.js, and a plain Webpack 5 setup.

The game concept:

- A high-school beginner coding and logic puzzle game.
- A fictional Japanese coastal reactor facility, explicitly not Fukushima Daiichi.
- A grid-based 3D/2.5D maze.
- The player selects a disaster-response robot.
- The player assembles command blocks, previews Python-like pseudocode, and runs or steps through the program.
- The robot seals fictional radiation hotspots using stabilizing foam and reaches extraction.
- The tone must remain arcade-puzzle, educational, and age-appropriate.

The full original implementation prompt is saved in:

- `prompts/init.md`

## Current Project State

The game is a runnable Webpack 5 React/TypeScript project.

Run locally:

```bash
npm install
npm start
```

Build and typecheck:

```bash
npm run typecheck
npm run build
```

Preview as Cloudflare Pages:

```bash
npm run pages:preview
```

Deploy by Wrangler direct upload:

```bash
npm run cf:whoami
npm run pages:deploy
```

The project intentionally keeps Cloudflare account-specific files ignored so the repository can be made public without committing local deployment state or secrets.
